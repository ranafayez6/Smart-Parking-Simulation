# Smart-Parking-Simulation

A smart parking simulation system built using **Python**, designed to simulate real-time parking management with intelligent pricing, queue management, payment processing, occupancy monitoring, and peak prediction.

## 📌 Project Overview

Smart Parking Pro MAX simulates how a modern parking system works in real life.

The system automatically:
- Generates different car types
- Parks vehicles across multiple floors
- Manages overflow using a queue system
- Calculates parking cost dynamically
- Processes simulated payments
- Predicts peak parking periods
- Tracks prediction accuracy
- Displays analytics and reports through graphs

This project combines **simulation, intelligent decision-making, analytics, and UI development** into one system.

---

##  Features

### 🚘 Smart Car Generation
The system generates different car types:
- **Normal Cars**
- **VIP Cars**
- **Electric Cars**

Each type has different parking pricing logic.

---

###  Multi-Floor Parking Management
- Two parking floors
- Fixed parking capacity
- Automatic parking allocation
- Queue system when parking is full

---

###  Dynamic Pricing System
Parking cost is calculated based on:

- Car type
- Parking duration
- Parking occupancy level
- Peak occupancy pricing
- Long-stay penalties

Examples:
- **VIP Cars** → Higher pricing
- **Electric Cars** → Discounted pricing
- **High Occupancy** → Increased pricing

---

###  Payment Processing Simulation
The system simulates different payment methods:

- Card Payment
- Wallet Payment
- Cash Payment

---

###  Occupancy Monitoring
Tracks parking occupancy over time to monitor:

- Parking utilization
- Congestion levels
- System performance

---

###  Peak Prediction System
The project predicts peak parking periods using queue and occupancy conditions.

It compares:
- **Predicted peak**
- **Actual peak**

Then calculates prediction accuracy.

---

###  Alert System
The dashboard automatically shows alerts when:

- Queue becomes too high
- Parking occupancy exceeds threshold

---

###  Analytics & Visualization
The system generates:

#### Occupancy Graph
Shows parking occupancy over time.

#### Queue Graph
Shows queue size over time.

#### Car Type Report
Pie chart visualization for:
- Normal cars
- VIP cars
- Electric cars

---

##  User Interface

The project includes a **Tkinter-based GUI dashboard** featuring:

- Start simulation

- View graphs
- Generate reports
- Live revenue tracking

- Real-time logs

- VIP vehicles generate higher revenue.
- Electric cars receive discounted pricing.
- Queue size can indicate congestion levels.
- Prediction accuracy improves with more simulation cycles.
